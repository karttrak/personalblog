Tools I use

Here are the tools I use to develop my site

Hosting

GitHub Pages

This is a free feature that GitHub offers. I went with this option because, well, it's free. It also allows me to update my site with my own code, which I did not have the freedom to do when using the free tier of Wordpress. If you are not a developer, there are simple themes you can use to get started. You also don't need to know how to code to add posts. As long as you can learn Markdown, which is a simplified formatting language that allows you to add headings, links, images, and style your text a little bit (among other things). This is enough to get you started. Anything more and you will probably want to pay for a more robust service that allows for more customizations without having to write code.
