Moving on!

Going forward

There are a few things I want to get done in August as well. These are the goals I have prioritized for one reason or another:

  • Add a Tools page (this is actually going out today)
  • Add a navigation menu on mobile
  • Look into using new fonts

Let's go over these briefly.

Tools page — it's another thing to go in the header and I've basically completed it already (at least the first iteration of it), so that's an easy one to check off.

Mobile navigation will be good to add because I am adding more to the desktop navigation, but I am just hiding them on mobile due to limited header real estate on small screens. I need to code up a solution for this because I don't want a reduced mobile experience.

As for new fonts, I've already done some exploration on that front. I just want to look at something different. I am not the biggest fan of Roboto Slab for my headings.

Finally, a CSS preprocessor would really ease up my styling workflow. I am already used to using SCSS from a prior job, and there are so many times I want to use mixins or variables, but I don't have access to those yet. I need to get this done. Plus, it would be good to have one shared folder for all my styling and then just compile everything into a single CSS file that I import into my HTML page.
